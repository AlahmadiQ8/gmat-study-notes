## Coordinates

### Slope

```
y = mx + b --> m = slope
Slope = (y2 - y1) / (x2 - x1)
```

### Distance Formula 

```
Distance = sqrt((x2 - x1)^2 + (y2 - y1)^2)
```

If lines are perpendiculr, then slopes are negative reciprocals
  
```
Line 1: y = 2x + 4
Line 2: y = -1/2 + 2
Line 1 is perpendicular to Line 2
```
